À propos de votre base de données

Votre base de données est un fichier SQLite (.db) stocké localement sur votre ordinateur, dans le dossier AppData de l'application. Toutes vos transactions sont enregistrées automatiquement à chaque ajout ou suppression.

Sauvegarde : Utilisez le bouton Exporter sauvegarde JSON régulièrement pour créer une copie de sécurité sur votre PC, clé USB ou cloud. En cas de perte, utilisez Restaurer pour tout récupérer.

Migration : Pour passer à un autre ordinateur, exportez le JSON, copiez-le sur le nouvel ordinateur, et importez-le.
